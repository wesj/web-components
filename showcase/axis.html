Axis

Description

An axis in a graph. Can be used to configure the bounds, grids, and labels.

A graph can contain multiple of the same axis direction. Data can specify which one they want to use with the "xAxis" or "yAxis" attributes.

Demos

Math.cos(x) Math.sin(x)

Attributes

min
Minimum value of the graph
max
Maximum value of the graph
values
A set of values to show along the graph
ticks
One of major, minor, or grid
labels
Whether or not to show labels on major grid lines or ticks

Styles

font
Controls font of labels
border
Controls the style of the main axis lines (color and width)
position
Controls the positioning of the axis. Supports "absolute"
right, left, top, bottom
Controls the positioning of an absolutely positioned axis. right or left for y-axis. top or bottom for x-axis.
