Axis

Description

An axis in a graph. Can be used to configure the bounds, grids, and labels.

Demos

Math.cos(x)

Attributes

min
Minimum value of the graph
max
Maximum value of the graph
values
A set of values to show along the graph
ticks
One of major, minor, or grid
labels
Whether or not to show labels on major grid lines or ticks

Styles

font
Controls font of labels
border
Controls the style of the main axis lines (color and width)
