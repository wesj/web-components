Line graph

Description

A graph that shows data, formatted as either markers or lines or both

Inside the node you can include data in JSON format. The graph will treat this as a series of key-value pairs, so if your data starts at zero and moves as integers for x-values, you can just use an array. Otherwise, use an object.

Future:

  • Make 2D arrays or x-y pairs an option.
  • Allow passing an object as a value, with color values or special annotations/labels for each data point

Demos
